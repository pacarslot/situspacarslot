# Robots.txt

## Deskripsi

Website PACARSLOT menggunakan file `robots.txt` untuk mengatur bagaimana mesin pencari seperti Google, Bing, dan crawler lainnya mengakses halaman website.

## Tujuan

- Mengizinkan mesin pencari melakukan crawling.
- Membantu proses indexing.
- Menyediakan lokasi sitemap XML.

## Konfigurasi

```txt
User-agent: *
Allow: /

Sitemap: https://pacarslot.com/sitemap.xml
```

## Lokasi

https://pacarslot.net/robots.txt

## Status

- Allow crawling
- Sitemap aktif
- Mendukung Google Search Console
